A list of R conferences and meetings

R User Groups

UseR groups are listed by contintent then by country. Where appropriate, useR groups are divided within country, e.g. in the USA, by state & in the United Kingdom, by member country.

The suggested format for listing a group is

  • Town name: Group name & link; @Twitter_handle

Funding

The R Consortium provides funding for local useR groups.
